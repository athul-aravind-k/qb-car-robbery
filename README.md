# qb-car-robbery
QBCore Framework based Car Robbery script Fivem
Created using PolyZone and QB-Taget

working
~~~~~~~~~~~
Car robbery will only start if mentioned cops are availabe and is onduty in the city.
once the player start the robbery all police in the city will get notified and will recive location in the map in an intervel of time (customizable in config).
if the Player Leave Car he will Get notified to get back inside the car within the time given in config, and if he fails to get back robbery will fail and car will be deleted.
After player reach the delivery location and deliver the vehicle he will get paid as per the amount in the config.
if a robbery is in progress another one cannot be started, player will be notified a robbery is in progress.
there is a customizable cooldown time between each robberies.
